# 取り扱い説明書

## 1. はじめに

このアプリは、収支の記録、予算管理、AI 相談、生活支援、購入アドバイスをまとめて行う家計簿です。

## 2. ログイン方法

利用できるログイン方法:

- Google
- LINE
- メール

ログイン後はホームからダッシュボードへ進みます。

## 3. 基本的な使い方

### 3.1 収支を入力する

1. 入力タブを開く
2. 種別を選ぶ
3. 金額、日付、カテゴリを入力する
4. 必要に応じてメモや固定費設定を入れる
5. 保存する

対応カテゴリ（支出）: 食費 / 住居 / 水道・光熱費 / 通信費 / 交通費 / 医療費 / 日用品 / 娯楽 / レジャー / 趣味 / 教育 / 自己投資 / 保険 / 税金 / 交際費 / サブスク / ペット / **美容・衣服** / **ゲーム** / **推し活** / 寄付・支援 / その他

### 3.2 ダッシュボードを見る

確認できる内容:

- 当月の収支
- 予算との差
- カテゴリごとの使い方
- 固定費見直し候補
- 安全度と生活レベル

### 3.3 予算を設定する

1. プリセット設定を開く
2. 収入や配分を入力する
3. 固定費、変動費、貯蓄、投資の割合を決める
4. 保存してダッシュボードに反映する

## 4. AI 機能の使い方

### 4.1 AI 分析

- 家計の状態を文章で確認したいときに使います

### 4.2 AI チャット

- 家計について質問したいときに使います

### 4.3 AI 店案内

1. 店種を選ぶ
2. エリアを入力するか、現在地検索を使う
3. 必要なら自由入力キーワードを入れる
4. 検索結果の一覧と地図を確認する

使いどころ:

- 食料品や日用品の買い物先を探したいとき
- 生活圏の候補店を整理したいとき
- 予算に合わせて買い物先を絞りたいとき

## 5. 購入アドバイザーの使い方

ダッシュボードの「目標」タブ内にある「購入アドバイザー」で大きな買い物の判断ができます。

### 5.1 基本の流れ

1. カテゴリを選ぶ（日用品 / 家電・スマホ / 美容・ファッション / ゲーム / 推し活 / 車 / 住宅）
2. 購入予定の金額を入力する
3. 「判定する」を押す

### 5.2 表示される情報

- **労働日数**: その金額を稼ぐのに必要な日数（手取り月収÷30 で計算）
- **貯蓄月数**: 毎月の貯蓄をそのまま続けた場合に貯まるまでの月数
- **購入可能ライン**: 今の収支で無理なく買えるかどうかの判定
  - 住宅・車はローン月額が収入の一定割合以内かどうかで判定
  - その他は月次余剰（収入−支出）と比較して判定

### 5.3 相場を調べる

- 「相場を調べる」欄に商品名や店名を入力して送信すると AI が市場価格の目安を教えてくれます

### 5.4 目標に追加する

- 「目標に追加」ボタンを押すと「目標」タブの個人目標リストに追加されます
- 追加後は進捗金額を更新して目標達成を管理できます

### 5.5 住宅・車のローン計算について

- 住宅: 35 年返済 / 年利 1% で月額を計算し、手取りの 25% 以内を目安にしています
- 車: 5 年返済 / 年利 3% で月額を計算し、手取りの 20% 以内を目安にしています
- あくまで目安です。実際のローン条件は金融機関にご確認ください

## 8. よくあるトラブル

### ログインできない

- ブラウザの Cookie を有効にする
- Auth0 / Google / LINE の設定値を確認する

### 現在地で店案内が使えない

- ブラウザのアドレスバー横のアイコンから位置情報を許可する
- 通信状況を確認する
- HTTPS 環境でのみ位置情報が利用可能

### エリア検索で店が見つからない

- 駅名や市区町村名に変える
- 自由入力キーワードを追加する
- 時間をおいて再試行する（OpenStreetMap サーバーが混雑している場合あり）

### 周辺のお店情報を取得できない

- しばらく待ってから再試行する
- 別の店種や小さめのエリアで試す

### 購入アドバイザーで金額が表示されない

- プリセット設定で手取り月収と月次貯蓄額を登録してください
- 登録後は再度「判定する」を押してください

### 保存できない

- セッション切れの可能性があります。再ログインしてください

## 9. 運用上の注意

- 実データを扱う場合は本番環境変数を必ず設定してください
- シークレットは `.env.local` または Vercel 環境変数で管理してください
- SQL migration 未適用だと一部機能が正しく動かない場合があります
