# 取り扱い説明書

## 1. はじめに

このアプリは、収支記録、予算管理、目標管理、AI 相談、生活支援をまとめて行う家計簿です。

## 2. 使い始める

### 2.1 ログイン方法

利用できるログイン方法:

- Google
- LINE
- メール

ログイン後はホームからダッシュボードへ進みます。

### 2.2 デモ表示

- UI だけ確認したい場合は Vercel の Preview Deployment を使います
- `/dashboard` にはダミーデータ付きのデモ表示があります

## 3. 基本操作

### 3.1 収支を入力する

1. 入力タブを開く
2. 種別を選ぶ
3. 金額、日付、カテゴリを入力する
4. 必要に応じてメモ、支払い方法、固定費設定を入れる
5. 保存する

対応カテゴリ（支出）:

- 食費 / 住居 / 水道・光熱費 / 通信費 / 交通費 / 医療費 / 日用品 / 娯楽 / レジャー / 趣味
- 教育 / 自己投資 / 保険 / 税金 / 交際費 / サブスク / ペット / 美容・衣服 / ゲーム / 推し活
- 寄付・支援 / その他

### 3.2 ダッシュボードを見る

確認できる内容:

- 当月の収支
- 予算との差
- カテゴリごとの使い方
- 固定費見直し候補
- 安全度と生活レベル
- 世代別の目標表示

### 3.3 予算を設定する

1. プリセット設定を開く
2. 収入や配分を入力する
3. 固定費、変動費、貯蓄、投資の割合を決める
4. 保存してダッシュボードに反映する

## 4. 目標タブの使い方

### 4.1 世代を切り替える

目標エリア上部で次を切り替えできます。

- 個人
- こども
- シニア

一般向けの詳細管理は「個人」表示内にあります。

### 4.2 ローン / 借入を管理する

1. 「ローン」タブを開く
2. 「ローンを追加」から名前、種別、借入総額、残高、月返済、年利を登録する
3. 毎月の返済時に「返済」を押す
4. 進捗バーと完済予定を確認する

### 4.3 先取積み立てを管理する

1. 「積立」タブを開く
2. 「先取積み立てを追加」を押す
3. 絵文字、名前、目標金額、現在額、目標月を入力する
4. 積み立てたら金額を追加する
5. 達成したものは過去へ移して整理する

### 4.4 個人目標を管理する

1. 「個人」タブを開く
2. 「個人目標を追加」から目標を作る
3. 手動加算かチケット加算で進捗を更新する
4. 達成したら過去へ移す

個人目標は、推し活、ご褒美、美容、趣味などの用途を想定しています。

### 4.5 固定費見直しを使う

1. 「固定費」タブを開く
2. 当月の固定費候補一覧を確認する
3. 内容を確認したら「確認済み」にする
4. 対応済みなら「Done & hide」で今月だけ非表示にする
5. 必要なら「今月非表示にした N 件を再表示」で戻す

表示される内容:

- 優先度
- 月額インパクト
- 発生理由
- 支払い方法
- 最終発生日
- 見直しアクション

### 4.6 徳ポイントとチケットを使う

1. 「徳」タブを開く
2. 合計ポイントと月別履歴を確認する
3. 獲得チケット数と利用可能数を確認する
4. 必要ならチケット価値を調整する
5. 個人目標タブに戻ってチケットを使う

チケットは徳ポイント `100` ごとに `1` 枚獲得できます。

## 5. 購入アドバイザーの使い方

「購入アドバイザー」で大きな買い物の判断ができます。

### 5.1 基本の流れ

1. カテゴリを選ぶ
2. 購入予定の金額を入力する
3. 判定を実行する

### 5.2 表示される情報

- 労働日数
- 貯蓄月数
- 購入可能ライン
- 車 / 住宅のローン月額目安

### 5.3 相場を調べる

- 商品名や店名を入れると AI が価格目安を返します

### 5.4 目標に追加する

- 「目標に追加」で個人目標タブへ送れます
- 追加後は目標タブで金額管理とチケット利用ができます

## 6. AI 機能

### 6.1 AI 分析

- 家計の状態を文章で整理したいときに使います

### 6.2 AI チャット

- 家計、節約、生活改善について会話できます

### 6.3 AI 店案内

1. 店種を選ぶ
2. エリア入力または現在地検索を使う
3. 必要に応じて自由入力キーワードを入れる
4. 候補一覧と地図リンクを確認する

## 7. 設定

設定画面では次を変更できます。

- デザイン
- アカウント情報
- 一部表示設定

## 8. 問い合わせ

問い合わせページから次を送信できます。

- 名前
- メールアドレス
- 件名
- 本文

## 9. よくあるトラブル

### ログインできない

- ブラウザの Cookie を有効にする
- Auth0 / Google / LINE の設定値を確認する

### 現在地で店案内が使えない

- 位置情報の許可を確認する
- 通信状況を確認する
- HTTPS 環境でのみ位置情報が利用可能

### エリア検索で店が見つからない

- 駅名や市区町村名に変える
- 自由入力キーワードを追加する
- 少し時間をおいて再試行する

### 月の入力 UI が出ない

- 一部ブラウザでは `input[type="month"]` の専用ピッカーが弱いことがあります
- `YYYY-MM` 形式で直接入力してください
- 迷う場合は Chrome / Edge 系ブラウザで確認してください

### 購入アドバイザーで金額が表示されない

- プリセット設定で手取り月収と月次貯蓄額を登録してください
- 登録後に再度判定してください

### 目標や徳ポイントが別端末で見えない

- ローン、積み立て、個人目標、徳ポイント関連の一部はブラウザの `localStorage` に保存されます
- 同じブラウザ / 同じ端末での利用を前提にしています

## 10. 運用上の注意

- 実データを扱う場合は本番環境変数を必ず設定してください
- シークレットは `.env.local` または Vercel 環境変数で管理してください
- SQL migration 未適用だと一部機能が正しく動かない場合があります
