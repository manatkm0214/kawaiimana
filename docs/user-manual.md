# 取り扱い説明書

## 1. はじめに

このアプリは、収支の記録、予算管理、AI 相談、生活支援をまとめて行う家計簿です。

## 2. ログイン方法

利用できるログイン方法:

- Google
- LINE
- メール

ログイン後はホームからダッシュボードへ進みます。

## 3. 基本的な使い方

### 3.1 収支を入力する

1. 入力タブを開く
2. 種別を選ぶ
3. 金額、日付、カテゴリを入力する
4. 必要に応じてメモや固定費設定を入れる
5. 保存する

### 3.2 ダッシュボードを見る

確認できる内容:

- 当月の収支
- 予算との差
- カテゴリごとの使い方
- 固定費見直し候補
- 安全度と生活レベル

### 3.3 予算を設定する

1. プリセット設定を開く
2. 収入や配分を入力する
3. 固定費、変動費、貯蓄、投資の割合を決める
4. 保存してダッシュボードに反映する

## 4. AI 機能の使い方

### 4.1 AI 分析

- 家計の状態を文章で確認したいときに使います

### 4.2 AI チャット

- 家計について質問したいときに使います

### 4.3 AI 店案内

1. 店種を選ぶ
2. エリアを入力するか、現在地検索を使う
3. 必要なら自由入力キーワードを入れる
4. 検索結果の一覧と地図を確認する

使いどころ:

- 食料品や日用品の買い物先を探したいとき
- 生活圏の候補店を整理したいとき
- 予算に合わせて買い物先を絞りたいとき

## 5. 設定

設定画面では次の内容を変更できます。

- デザイン
- アカウント情報
- 一部表示設定

## 6. 問い合わせ

問い合わせページから次を入力して送信できます。

- 名前
- メールアドレス
- 件名
- 本文

## 7. よくあるトラブル

### ログインできない

- ブラウザの Cookie を有効にする
- Auth0 / Google / LINE の設定値を確認する

### 現在地で店案内が使えない

- ブラウザのアドレスバー横のアイコンから位置情報を許可する
- 通信状況を確認する
- HTTPS 環境でのみ位置情報が利用可能

### エリア検索で店が見つからない

- 駅名や市区町村名に変える
- 自由入力キーワードを追加する
- 時間をおいて再試行する（OpenStreetMap サーバーが混雑している場合あり）

### 周辺のお店情報を取得できない

- しばらく待ってから再試行する
- 別の店種や小さめのエリアで試す

### 保存できない

- セッション切れの可能性があります。再ログインしてください

## 8. 運用上の注意

- 実データを扱う場合は本番環境変数を必ず設定してください
- シークレットは `.env.local` または Vercel 環境変数で管理してください
- SQL migration 未適用だと一部機能が正しく動かない場合があります
